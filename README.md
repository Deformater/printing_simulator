<h1>Тренажёр для печати</h1>
Представляю вам тренажёр для печати, написанный на PyQT5.<br>
Данный тренажёр помагает увеличить скорость печати, и уменьшить кол-во ошибок при быстрой печати.<br>
Этот тренажёр предлагает встроенный набор текстов на русском и английском языке. Но вы также можете добавить свой собственный текст.<br>
Данные о скорости печати хранятся в удобной базе данный с возможностью фильтровать данные по времени.
